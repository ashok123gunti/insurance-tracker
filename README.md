# 🛡️ Insurance Tracker — Web Application

A clean, responsive web application built to help users track and manage their insurance policies with ease. Focused on delivering an optimised user experience with seamless navigation and dynamic form validation.

---

## 🔍 About the Project

Insurance Tracker is a front-end web application that simplifies the process of managing multiple insurance policies. Users can add, view, and track their insurance details through an intuitive interface built with core web technologies — HTML, CSS, and JavaScript.

---

## ✨ Features

- 📋 **Policy Management** — Add and track multiple insurance policies in one place
- ✅ **Dynamic Form Validation** — Real-time input validation to reduce data entry errors
- 📱 **Responsive Design** — Seamlessly adapts to desktop and mobile screens
- 🧭 **Smooth Navigation** — Intuitive UI with clean page transitions
- ⚡ **Lightweight** — Pure HTML/CSS/JS — no frameworks, no dependencies

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure and markup |
| CSS3 | Styling and responsive layout |
| JavaScript (ES6) | Logic, form validation, interactivity |

## 🚀 Getting Started

### Option 1 — Open Directly
Simply clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/ashok123gunti/insurance-tracker.git
cd insurance-tracker
# Open index.html in your browser
```

### Option 2 — Live Server (Recommended)
If you have VS Code, use the **Live Server** extension for the best experience:
1. Open the project folder in VS Code
2. Right-click `index.html` → **Open with Live Server**
